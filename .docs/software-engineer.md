# Software Engineer for Top Dutch Solar Racing

As a Software Engineer, you connect and coordinate all electrical systems so the solar car operates seamlessly. You'll work close to the hardware on microcontrollers, build reliable CAN communication, and deliver software that keeps the vehicle controllable and dependable. Embedded firmware from low-level drivers to race-ready reliability.

More info at https://www.solarracing.nl

## Highlights

- MCU firmware (C/C++) and CAN stacks
- Reliability, diagnostics, and safety
- Testing and validation

## What will you do

- Connect and coordinate all the electrical systems of the solar car to ensure that it operates seamlessly
- Implement and maintain low-level firmware (drivers, interrupts, timers) and robust CAN communication between subsystems (e.g., BMS, lighting, steering).
- Improve existing code and implement new functionalities.
- Collaborate closely with Strategy and Electronics.
