# Batter Management System GUI

An MVP of a real-time monitoring interface for solar car BMS systems that visualizes cell voltages, temperatures, and other metrics.

## Demo video

There is a short demo video [here](https://www.loom.com/share/73be8d4853bb4efd965ec9feb48e11be).

## Building the program

A pre-built executable for Linux is available at the [Releases](https://github.com/petdomaa100/solar-racing-assignment/releases) page.

```bash
mkdir .build

cd .build

cmake ..

make

./bms_gui
```
